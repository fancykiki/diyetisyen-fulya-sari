# Useful OpenCode Prompts — Diyetisyen Fulya Sarı

## 1. First Analysis
Projeyi AGENTS.md kurallarına göre analiz et. Özellikle ana sayfa hero/slider alanını, responsive yapıyı, header/navbar ve CTA bölümlerini incele. Kod yazmadan önce kısa bir revize planı çıkar ve hangi dosyalara dokunacağını söyle.

## 2. Safe Hero Improvement
Mevcut ana slider/hero tasarımını bozmadan daha premium, modern ve responsive hale getir. Sadece hero/slider ve gerekli CSS/JS dosyalarına dokun. Kartı daha kompakt ve zarif yap, feature pill hizalamalarını düzelt, CTA butonlarını güçlendir, arka plan overlay dengesini iyileştir, slider navigation alanını sadeleştir. Mobilde taşma olmasın. Sonunda değiştirdiğin dosyaları raporla.

## 3. Animation Pass
Hero/slider alanına sade ve premium animasyonlar ekle. Başlık, açıklama, feature pill ve CTA butonları sırayla yumuşak şekilde gelsin. Arka plan fotoğrafında çok hafif slow zoom olabilir. Animasyonlar CSS ile yapılabiliyorsa dependency ekleme. `prefers-reduced-motion` desteği ekle. Mobilde animasyonları sadeleştir.

## 4. Mobile Fix Pass
Projede 1440, 1024, 768, 480 ve 375px genişliklerde responsive sorunları kontrol et. Header, hero card, slider navigation, feature pill, CTA butonları ve footer alanlarında taşma/kırılma varsa minimal CSS ile düzelt. Gereksiz dosyalara dokunma.

## 5. UI Lab Repo Prompt
Bu repo, Diyetisyen Fulya Sarı web sitesi için UI/UX tasarım laboratuvarıdır. 5 farklı hero/slider konsepti üret: Premium Nutrition, Online Consultation, Healthy Lifestyle Journey, Meal Planning, Weight Management. Lottie, GSAP ve hafif Three.js/React Three Fiber denemeleri yapabilirsin; ama componentler ana projeye kolay taşınabilir olsun. Mobile performance önemli. Her varyantı ayrı component yap ve sonunda en uygun varyantı öner.
